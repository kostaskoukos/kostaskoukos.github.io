# Kostas Koukos' Portfolio Website

This is my website's public repository. It is hosted on Github Pages at [https://kostaskoukos.github.io](https://kostaskoukos.github.io).

## Content

My Portfolio contains a few things about me, some notable projects I have worked
on and their github/live website links, blog posts about Computer Science
or anything off-topic,
and social media handles/how to contact me.

## Run locally

This project is a static website generated by [Astro.js](https://astro.build)
with [TailwindCSS](https://tailwindcss.com) and its [typography plugin](https://github.com/tailwindlabs/tailwindcss-typography).
